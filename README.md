# 🎲 Random Image Gallery (Picsum)

A clean and fun React app that displays random images using Picsum Photos.

> ⚡ No API key. No backend. Just pure randomness.

---

## ✨ Features

- 🎲 Generates random images every time
- 🖼️ Beautiful responsive image grid
- ⚡ Lightweight and fast
- 🔄 New images on refresh
- 🧑‍💻 Beginner-friendly React project

---

## 🛠️ Tech Stack

- ⚛️ React
- 🎨 Tailwind CSS
- 🌐 Picsum Photos API
